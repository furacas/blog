这些指令不需要死记，用的时候查表知道是干什么的就可以了。

| **指令** | **作用**                  |
| -------- | ------------------------- |
| cmp a,b  | 比较 a,b 大小             |
| mov a,b  | b 的值赋给 a              |
| ret      | 返回主程序                |
| nop      | 空语句                    |
| call     | 调用子程序                |
| je/jz    | 相等则跳转 机器码是 74/84 |
| jez/jnz  | 不相等则跳 机器码是 75/85 |
| jmp      | 无条件跳 机器码是 EB      |
| jb/jl    | 小于则跳                  |
| ja/jg    | 大于则跳                  |
| jge      | 大于等于则跳              |
| pop xxx  | xxx 出栈                  |
| push xxx | xxx 压栈                  |

如果想了解更详细可以参考一下[逆向汇编基础 ](https://www.beimengyeyu.com/2022/kq0iqk/)
